## 🚀 Featured Project

### 🛡️ [AI CCTV Threat Monitor](https://github.com/KAN915A/ai-cctv-threat-detection)

Real-time security monitoring powered by a **3-model YOLOv8 ensemble** that classifies threats
into four alert levels — loitering, vehicle lurking, weapons, and altercations.

<p>
  <img src="https://img.shields.io/badge/YOLOv8-Ensemble-FF6B6B?style=flat-square" />
  <img src="https://img.shields.io/badge/FastAPI-Backend-009688?style=flat-square&logo=fastapi&logoColor=white" />
  <img src="https://img.shields.io/badge/ONNX%20Runtime-In--Browser-005CED?style=flat-square&logo=onnx&logoColor=white" />
  <img src="https://img.shields.io/badge/PyTorch-Inference-EE4C2C?style=flat-square&logo=pytorch&logoColor=white" />
  <img src="https://img.shields.io/badge/Deployed-Vercel-000000?style=flat-square&logo=vercel&logoColor=white" />
</p>

- 🧠 **Custom-trained weapon detectors** (mAP50 ~0.86) fused with a COCO context model
- 🎯 A **smart fusion filter** kills false positives — size vetoes, person-coincidence checks, and tiered confidence gates
- 🌐 Two editions from one model core: a **Python + FastAPI server** (RTSP, SQLite history, Telegram push alerts) and an **in-browser ONNX demo** where video never leaves your device
- ⚡ **[Live Demo →](https://ai-cctv-threat-detection.vercel.app)**
